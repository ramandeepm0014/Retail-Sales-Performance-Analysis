# 📊 Retail Sales Performance Analysis Dashboard

A professional **Retail Sales Performance Analysis Dashboard** developed entirely in **Microsoft Excel**. This project demonstrates data analysis, interactive dashboard creation, and business insights using Excel features such as Pivot Tables, Pivot Charts, KPIs, and Slicers.


## 📌 Project Overview

This project analyzes retail sales data to identify sales trends, product performance, and key business metrics. The dashboard provides an easy-to-understand visual representation of sales performance, helping users make data-driven decisions.


## ✨ Features

- 📊 Interactive Excel Dashboard
- 📈 Sales Performance Analysis
- 📋 Pivot Tables
- 📉 Pivot Charts
- 🎯 KPI Metrics
- 🎛️ Interactive Slicers
- 📂 Organized Excel Workbook
- 💡 Business Insights


## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Charts
- Data Cleaning
- KPI Dashboard


## 📁 Workbook Contents

The Excel workbook contains:

- **Retail Sales Dashboard**
- **All Tables and Graphs**
- **Retail Sales Dataset**


## 📊 Dashboard Highlights

The dashboard provides insights into:

- Overall Sales Performance
- Sales Trends
- Product Performance
- Business KPIs
- Interactive Filtering using Slicers
- Visual Reports using Charts


## 📂 Dataset

The project is based on a retail sales dataset stored in the Excel workbook. The data has been organized and analyzed to create meaningful reports and dashboards.


## 🎯 Project Objectives

- Analyze retail sales performance.
- Create an interactive Excel dashboard.
- Visualize business data using Pivot Charts.
- Generate business insights using KPIs.
- Improve decision-making through data analysis.
